Usage
-------

### Install ###

    $ git clone git@github.com:otto-schnurr/AdventOfCode.git
    $ cd AdventOfCode/2020

### Test ###

    $ swift test && echo "Solutions have been verified."

### Browse ###

With Xcode 12 or later.

    $ open Package.swift
